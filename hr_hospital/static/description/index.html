HR Hospital

HR Hospital — це модуль для Odoo, який автоматизує роботу медичних закладів. Дозволяє вести облік пацієнтів, лікарів, інтернів, візитів, діагнозів та класифікатора хвороб.

Основні можливості:

  • Облік пацієнтів, лікарів, інтернів, менеджерів та адміністраторів
  • Планування та реєстрація візитів пацієнтів
  • Ведення історії діагнозів
  • Ієрархічний класифікатор хвороб
  • Гнучка система доступу (пацієнт, інтерн, лікар, менеджер, адміністратор)
  • Звіти та аналітика
  • Підтримка української локалізації

Для кого цей модуль?

  • Медичні центри, клініки, приватні лікарі
  • Менеджери та адміністратори медичних установ
  • Розробники рішень для Odoo у сфері медицини
